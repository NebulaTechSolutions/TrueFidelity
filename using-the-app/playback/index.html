    Playback¶

    Playback is the heart of TrueFidelity. This guide details timeline navigation, transport controls, range selection, smart filtering, and tips for handling large log files.

    Transport Controls¶

    • Play/Pause (Space) – Start or stop playback.
    • Step Forward/Backward (→/←) – Move one frame at a time. Hold Shift to jump one second.
    • Jump to Start/End (Home/End) – Quickly reset to the beginning or skip to the end.
    • Speed Selector – Choose from 0.25×, 0.5×, 1×, 2×, 4×, 8×. Speeds above 4× rely on your system’s I/O performance.
    • Loop Toggle – When enabled, the selected range repeats until you disable it.
    • Sync Playheads – If multiple charts are open, enabling sync keeps all charts aligned with the timeline.

    Tip

    Assign custom shortcuts to common speeds (e.g., 1×, 2×, 0.25×) under Settings → Shortcuts.

    Timeline Navigation¶

    • Click the timeline to move the playhead instantly.
    • Drag to select a range. The status bar shows start, end, and duration.
    • Scroll wheel zooms in/out of the cursor position. Hold Ctrl/Cmd while scrolling for slower zooming.
    • Shift + Scroll pans left/right without changing zoom.
    • Double-click the timeline background to reset zoom to full duration.
    • Markers appear for annotations, bookmarks, or injection events. Hover to see tooltips.

    Range Selection Workflows¶

    • To focus on an event, drag to highlight a range and enable Filter to Range (toolbar button). Messages, charts, and statistics now show only data inside the selection.
    • Use Loop to replay the selected range while adjusting filters or annotations.
    • Right-click the range to save it as a named segment (e.g., ABS Event 1). Named segments help with quick navigation and exports.

    Messages View Sync¶

    • Messages automatically scroll to match the current timeline position.
    • Apply filters (ID, payload, channel, time window) to reduce noise.
    • Freeze Scroll to examine a section while playback continues. When ready, unfreeze to re-sync.

    Charts & Signals During Playback¶

    • Pinned signals appear in the chart. Drag additional signals to compare behavior.
    • Use Auto-scale for exploratory work and Fixed scale when producing reports for consistent axes.
    • Add Cursors (Shift + C) to measure time differences between two points. The delta displays in the chart legend.
    • Toggle series visibility using the legend to isolate signals temporarily.

    Annotations & Bookmarks¶

    • Press A to create an annotation at the current playhead. Add tags, descriptions, and severity (if configured).
    • Use Bookmarks (B) for quick markers without extra metadata.
    • Annotated points show directly on the timeline; clicking an annotation jumps to the location and opens details in the right-hand panel.

    Counters & Diagnostics¶

    • Messages/sec – Real-time throughput for the current view. High values indicate denser sections of the log.
    • Dropped frames – Highlights hardware or decoding bottlenecks. Investigate when non-zero.
    • CPU/IO gauges – Optional overlay showing resource usage; enable in Settings → Diagnostics.

    Handling Large Logs¶

    • Load only the logs you need for a session. Remove others using the project explorer context menu.
    • Use Filter-to-Range and message filters to avoid rendering millions of frames at once.
    • Reduce chart complexity—combine signals into fewer charts when possible.
    • Store logs on SSDs or NVMe drives to avoid I/O bottlenecks.
    • Enable Incremental Decoding (Settings → Data) to decode as you scrub, reducing initial load times.

    Working with Multiple Logs¶

    • Enable Multi-log mode to overlay multiple runs. Each log displays with a different color band on the timeline.
    • Use Align by Timestamp or Align by Offset options to synchronize logs.
    • Mute individual logs temporarily using checkboxes in the project explorer.

    Playback for Live Streams¶

    If your deployment supports live capture:

    • Use the same controls; the timeline shows a trailing buffer for live data.
    • Loop mode is disabled while live streaming; manual range selection still works for captured segments.
    • Annotations are supported; they bind to the recorded timestamp once data is flushed to disk.

    Exporting from Playback¶

    • Export Segment – Uses the currently selected range. Choose format (CSV, MDF4) and destination.
    • Screenshot – Use the camera icon in Charts or the Annotations panel to capture context.
    • Report Builder – While paused, launch the builder to queue charts, annotations, and notes into a report template.

    Keyboard Shortcut Reference¶

    Action Shortcut
    Play/Pause Space
    Step forward/backward → / ←
    Jump ±5 seconds Shift + → / Shift + ←
    Jump to start/end Home / End
    Toggle loop L
    Add annotation A
    Add bookmark B
    Toggle filter-to-range Ctrl/Cmd + Shift + F
    Zoom in/out = / - (or mouse wheel)

    Troubleshooting Playback¶

    Problem Suggested Fix
    Playback stutters Reduce speed, close unused panels, move logs to faster storage.
    Charts lag behind Limit to critical signals, disable auto-scale, or increase chart buffer in Settings → Charts.
    Time jumps unexpectedly Check for large gaps or overlapping timestamps; use Normalize timestamps option in the timeline menu.
    No audio cues Some builds include optional audio alerts; enable under Settings → Accessibility.

    Best Practices¶

    • Use consistent naming for annotations and segments to streamline reporting.
    • Pair playback with filters; start broad, then refine as you zero in on anomalies.
    • Save snapshots of critical ranges so you can return during reviews without rescanning the timeline.
