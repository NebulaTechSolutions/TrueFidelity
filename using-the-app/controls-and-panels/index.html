    Controls & Panels¶

    Understanding each panel and control allows you to tailor TrueFidelity to your workflow. This guide describes panel functionality, configuration options, and best practices.

    Panels are organized by workflow: System Design, ECU Monitoring, and CAN Log Analysis. Most panels are dockable and can be rearranged to match your preferred layout.

    System Design Panels¶

    System Canvas¶

    • Drag-and-drop: Add ECUs from the Toolbox by dragging onto the canvas
    • Network connections: Click and drag between ECUs to create network edges
    • Multi-select: Hold Ctrl/Cmd and click multiple elements, or drag a selection box
    • Zoom controls: Mouse wheel to zoom, or use toolbar buttons
    • Pan: Middle-click and drag, or spacebar + drag
    • Export: Right-click canvas to export system configuration

    Tip

    Use the grid snap feature (View → Snap to Grid) to keep your layouts clean and organized.

    Toolbox¶

    • Categories: Browse ECU types, network interfaces, and gateway components
    • Search: Type to filter available components
    • Drag to add: Drag any component from Toolbox to Canvas to instantiate
    • Templates: Save custom ECU configurations as templates for reuse

    Properties Panel¶

    • Context-aware: Shows properties for the currently selected element (ECU, network, edge)
    • Editable fields: Click any property value to edit directly
    • Validation: Invalid values are highlighted with error indicators
    • Presets: Save property configurations as presets

    Project Explorer¶

    • File tree: Navigate system files, builds, and configurations
    • Context menu: Right-click for options (open, delete, rename)
    • Favorites: Pin frequently accessed files for quick access
    • Search: Use Ctrl/Cmd + F to search within the explorer

    ECU Monitoring Panels¶

    ECU Status Panel¶

    • Grid view: All ECUs displayed in a sortable table
    • Status indicators: Color-coded states (green=running, yellow=starting, red=error, gray=stopped)
    • Quick actions: Right-click any ECU for context menu (start, stop, restart, logs)
    • Resource summary: Hover over resource cells to see detailed metrics
    • Double-click: Open ECU detail view in new panel

    Console Panel¶

    • Unified logs: Aggregated output from all ECUs in one view
    • Filter bar: Filter by ECU name, log level, or search term
    • Log levels: Toggle visibility of info, warn, error, debug messages
    • Color coding: Log levels are color-coded for quick scanning
    • Auto-scroll: Toggle to keep view locked to latest messages
    • Export: Export filtered logs to text file

    Tip

    Use the pause button to freeze the log view while investigating a specific message, then resume auto-scroll.

    Resource Monitor¶

    • Real-time graphs: Live CPU and memory usage over time
    • Per-ECU metrics: Separate lines for each ECU with color coding
    • Time range: Adjust visible time window (1min, 5min, 15min, 1hr)
    • Zoom: Click and drag on graph to zoom into specific time ranges
    • Export: Export metrics as CSV for analysis or PNG for reports

    Network Traffic Panel¶

    • Live monitoring: Real-time CAN message stream
    • Filters: Filter by message ID (hex or decimal), channel, or source ECU
    • Message details: Click any message to see payload, timestamp, flags
    • Rate statistics: View messages per second and total counts
    • Export: Capture traffic to log file (BLF, MDF4, CSV)
    • DBC integration: Signals are decoded when DBC files are loaded

    Terminal Panel¶

    • Multi-tab interface: One terminal tab per ECU
    • Full shell access: Interactive terminal to ECU operating system
    • Connection status: Visual indicators show connection state
    • Search: Press Ctrl+F to search terminal output
    • Terminal actions: Clear screen, copy selection, reconnect
    • Resize handling: Terminal automatically adjusts to panel size

    Warning

    Terminal access requires appropriate permissions. Some ECUs may restrict root access or certain commands.

    Process Monitor¶

    • Process list: View all running processes within selected ECU
    • Resource usage: CPU and memory per process
    • Process tree: Expandable tree view showing parent-child relationships
    • Sortable columns: Click column headers to sort
    • Refresh rate: Adjust update frequency in panel settings

    CAN Log Analysis Panels¶

    Signals Panel¶

    • Search: Filter by name, ID, unit, or tags. Partial matches supported.
    • Pinned signals: Click the pin icon to add to the watch list; pinned signals persist across sessions.
    • Context menu: Right-click a signal to add to charts, copy name, or highlight in Messages.
    • Grouping: Use tags or DBC categories to sort signals. Toggle groups in the left-hand tree.
    • Derived signals: If configured, derived entries appear with a calculator icon.

    Messages Panel¶

    • Columns: Timestamp, ID, DLC, payload, channel, flags. Choose Columns from the toolbar to show/hide fields.
    • Freeze scroll: Lock the view while playback continues—useful for inspecting a burst without losing context.
    • Color rules: Apply highlight rules to emphasize frames matching filters (e.g., error flags).
    • Export: Use Export Visible to write a CSV of the filtered dataset.

    Charts Panel¶

    • Add signals: Drag from the Signals panel or use the + button inside Charts.
    • Axes: Right-click to split signals across multiple Y-axes or lock them to a shared scale.
    • Cursors: Press Shift + C to add cursors; the delta time and value difference show in the legend.
    • Legends: Toggle series visibility; double-click a legend entry to isolate a signal temporarily.
    • Screenshots: Capture a PNG directly from the toolbar; stored in your export folder.

    Timeline¶

    • Selection: Drag to select; double-click to reset.
    • Markers: Bookmark and annotation markers appear; click to open details.
    • Named segments: Right-click and save selections with descriptive names for reuse.
    • Zoom: Scroll wheel zooms; Shift + wheel pans. Keyboard = and - also zoom.

    Annotations Panel¶

    • Lists bookmarks and annotations chronologically.
    • Filter by tags, severity, or author.
    • Export as CSV or send to Report Builder.
    • Templates accelerate repetitive annotation entry.

    Data Injection Panel¶

    • Message list: Shows defined templates with ID, payload, schedule, status.
    • Schedule editor: Configure once, interval, burst, or ramp profiles.
    • Counters: Track sent/failed frames, rate, and elapsed time.
    • Profiles: Save multiple injection scenarios; export/import for consistency across test stations.

    Watch List¶

    • Pinned signals appear here for quick reference.
    • Drag watch entries to Charts to plot or to Annotations to insert current values into notes.
    • Reorder using drag-and-drop to keep key signals on top.

    Filters Panel¶

    • Manage saved filters and filter sets.
    • Toggle filters on/off without deleting them.
    • See active filters at a glance; apply them to Messages, Charts, or both.

    Export Queue¶

    • View and manage pending exports.
    • Reorder, pause, or delete queued tasks.
    • Useful when batch exporting multiple segments overnight.

    Reports Panel¶

    • Launch Report Builder.
    • Assemble annotations, charts, statistics, and screenshots into PDF/HTML packages.
    • Preview report sections before exporting.

    Properties/Inspector Panel¶

    • Shows metadata for the selected item (frame, signal, annotation, export task).
    • Edit description, tags, or destination paths directly.
    • Displays derived values such as scaling and raw hex.

    Layout Controls¶

    • Docking handles appear when dragging panel tabs.
    • Split view icons split panels vertically or horizontally (in some builds).
    • Reset layout returns everything to defaults; use saved layouts to restore custom arrangements.

    Panel Shortcuts¶

    Panel Shortcut (default) Context
    ECU Status / Signals Ctrl/Cmd + 1 Monitoring / Analysis
    Console / Messages Ctrl/Cmd + 2 Monitoring / Analysis
    Resource Monitor / Charts Ctrl/Cmd + 3 Monitoring / Analysis
    Terminal / Data Injection Ctrl/Cmd + 4 Monitoring / Analysis
    Annotations Ctrl/Cmd + 5 Analysis
    Filters Ctrl/Cmd + 6 Analysis
    Project Explorer Ctrl/Cmd + B All modes
    System Canvas Ctrl/Cmd + Shift + C Design
    Properties Ctrl/Cmd + Shift + P All modes

    Customize these shortcuts under Settings → Shortcuts to align with your preferences. Some shortcuts are context-dependent and switch functionality based on the active workflow mode.

    Panel Tips¶

    • Focus mode: Double-click a panel tab to expand it to full screen; double-click again to restore.
    • Linked selections: Enable linked selection between Messages and Charts so selecting a frame highlights corresponding signal values.
    • Panel presets: Save panel configurations (e.g., chart axis settings) as presets to reapply later.

    Troubleshooting Panels¶

    Problem Solution
    Panel missing Check if it’s minimized or tabbed behind another panel. Use View menu to reopen.
    Panel flickers Disable hardware acceleration via launch flag or update GPU drivers.
    Panel content blank Reset layout; ensure filters aren’t hiding all data.
    Dragging unintentionally undocks panels Hold Alt while dragging to prevent undocking, or adjust drag thresholds in settings.
